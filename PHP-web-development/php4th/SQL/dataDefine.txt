SQL语言的数据定义

1.基本表的定义：
	标的关系框架用SQL语句定义
	
CREATE TABLE S(
				SN CHAR(10) NOT NULL,LookupError
				NAME CHAR(10),
				AGE SMALLINT,
				SEX CHAR(1),
				PRIMARY KEY(SN), CHECK(AGE BETWEEN 15 AND 25));
				
CREATE TABLE C(
				CN CHAR(6) NOT NULL,
				TIME SMALLINT NOT NULL,
				TEACHER CHAR(10),
				OFFICE CHAR(4),
				PRIMARY KEY(CN));
				
CREATE TABLE SC(
				SN CHAR(7) NOT NULL,
				CN CHAR(6) NOT NULL,
				GRADE DEC(4, 1) DEFAULT NULL,
				PRIMARY KEY(SN, CN),
				FOREIGN KEY(SN) REFERENCE S ON DELETE CASCADE,
				FOREIGN KEY(CN) REFERENCE C ON DELETE RESTRICT);
				
属性如下：
SN 学号、NAME 姓名、AGE 年龄、SEX 性别、CN 课程号、TIME 学时、GRADE 成绩、TEACHER 任课教师、OFFICE 办公室
S 学生关系表
	SN	TIME	AGE		SEX
C 课程关系表
	CN	TIME	TEACHER	OFFICE
SC 学生选课关系表
	SN	CN		GRADE
	
在基本表中增加一列 ALTER
ALTER TABLES ADD (TELN CHAR(20))
语句执行后在 S表 中最后增加一列 TELN

撤销已存在的表 DROP
DROP TABLE SC


2.建立和撤销视图
	定义一个共打印成绩表所用的视图
CREATE VIEW STUDENT_GRADE
AS SELECT S.SN, S.NAME, C.CN, GRADE
FROM S, C, SC
WHERE S.SN = SC.SN AND SC.CN = C.CN
	DROP 语句撤消视图
DROP VIEW STUDENT_GRADE
